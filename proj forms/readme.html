How-To... - Launch Files with their Associated Applications through Visual Basic

When using the Shell function through Visual Basic, if you wish to launch, for example, a .DOC file. You would expect it to open up and load into Wordpad or Word, but it doesn't. The Shell function can only be used to run EXE, BAT, COM, etc. To be able to launch other files such as DOC, or TXT, you have to use the ShellExecute API call. The call is as follows :-

Private Declare Function ShellExecute Lib "shell32.dll" Alias "ShellExecuteA" _
(ByVal hwnd As Long, ByVal lpOperation As String, _
ByVal lpFile As String, ByVal lpParameters As String, _
ByVal lpDirectory As String, ByVal nShowCmd As Long) As Long

This "How-To" shows you how you can launch files with their associated applications.
